# Louvre-Heist-Story-Website
This was a team-based academic project. The website is designed as an interactive digital story about the Louvre Heist. The homepage introduces the theme using a looping background video, audio controls, animated content cards, and article-style aftermath links. The process page presents the timeline of the heist through slide-based storytelling, while the loot page displays stolen and recovered items in a gallery format. The website also includes a contact page and a references page.

## Features
- Multi-page website structure
- Responsive navigation bar with hamburger menu
- Video-based hero section
- Background audio with play/pause control
- Scroll-based navbar behavior
- Animated content cards using JavaScript and Intersection Observer
- Interactive story slides with previous/next controls
- Keyboard navigation for story slides
- Loot gallery displaying stolen and recovered items
- Contact form
- References page for sources and design inspiration
- Consistent theme, layout, and visual styling


## Pages
- `index.html` - Homepage with video hero section, background music, introduction, escape video, and aftermath cards
- `process.html` - Interactive story page explaining the process and timeline of the heist
- `loot.html` - Gallery page showing the stolen and recovered items
- `contact.html` - Contact form page
- `references.html` - Sources and design references
